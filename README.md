# Cross-Asset Derivatives Intelligence Platform

Minimal starter for a Streamlit portfolio project focused on cross-asset derivatives analysis.

## What’s here

- `app.py`: Streamlit landing page
- `src/`: space for application code
- `tests/`: space for tests

## Run it

```bash
python -m venv .venv
.venv\\Scripts\\python -m pip install -r requirements.txt
.venv\\Scripts\\streamlit run app.py
```
